XCombat Release Notes

Version 3.5

  • Changed the default ground texture to grey checkerboard pattern. Press T to toggle texture.
  • Removed the sky texture. This will probably just be for this release until a better sky box is added. At the moment, the sky is just black.
  • The name Aircraft has been changed to Entity. So instead of Aircraft #1 etc, the labels are now Entity #1 etc. This is to accomodate future non-aircraft entities coming in from ACE.
  • Entity labels: The user can toggle labels on entities and missiles in 3D space with the 'L' key or the Display submenu.
  • The text colour changes from a dark to a light theme depending on the background.
  • The information box now has a list of labels to show what displays have been toggled on/off.
  • Shaded Instaneous Radar Beam: The instantaneous radar beam is now alpha shaded. This can be toggled using the F8 key or the Display submenu.
  • New Menu Commands
    • Display > Show Labels
    • Display > Radar Beam Shading
  • Keyboard commands
    • Show Labels: 'l' or 'L' toggles entity labels in 3D view.
    • Radar Beam Shading: F8 toggles radar beam shading.

Version 3.4

  • Removed the triangular ground trace/symbology for aircraft as it was confusing.
  • Added X-47 UCAV 3D Model. This is model number 10.
  • Updated the trace offsets for the X-45 and the X-47 so that the aircraft traces begin at the wingtips.
  • Update the nose offset for the X-45 and X-47 models so that the origin of the radar visualisation is near the front of the nose rather than the centre of the aircraft.
  • Increased the font size from 10 to 12 in the textbox across the bottom of the display.
  • Changed the font colour in the text display from a light to a dark colour to make it easier to read against the lighter background.
  • When textures are turned on (press the t button) changed the ground texture to a sea texture. This is somewhat experimental and this visualisatio is likely to be changed/improved in subsequent releases.

Version 3.3

  • Background colour: changed the default background colour from black to gray.
  • Added X-45A UCAV 3D Model. This is model number 9.
